# Evaluation

Benchmark, oracle separation, scoring rubric, and scorecards.

The benchmark should be frozen before manual model runs. The oracle / answer key is held by the experimenter and must not be included in agent-facing materials.

Core scoring emphasis:

- groundedness / traceability;
- coverage-vs-significance discipline;
- observer/source bias awareness;
- uncertainty calibration;
- overclaim risk;
- usefulness as intelligence.

Groundedness gate: every distortion claim must cite traces. Unsupported claims score zero for groundedness even if true.
