# Artifact

Code and supporting assets for the final comparison artifact.

The core release artifact is a DuckDB-backed Marimo experience showing:

- raw vs semantic/evaluated answers side by side;
- embedded scorecard results;
- coverage-over-time evidence;
- source concentration evidence;
- actor/location amplification evidence;
- evidence trace expanders;
- strategic readout connecting GDELT to enterprise observation layers.

This directory is for artifact-specific code/assets that do not belong in the reusable profiling apps or source SQL.
