# Prompts

Controlled manual-run materials for the four experimental conditions.

Planned prompt packs:

1. `parametric-only` — no data;
2. `raw-data` — raw/staged GDELT data plus neutral schema docs only;
3. `semantic` — raw data plus lens guides;
4. `evaluated` — semantic condition plus groundedness, uncertainty, and evidence requirements.

This directory will also hold transcript templates and leakage-boundary notes for manual runs.
